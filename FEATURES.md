# Harts Beauty

## Features

Web application has the following pages:
- home page
- about page
- Treatment detail page
- login page
- registration page
- logout page
- make booking page
- booking confirmation page
- my bookings page
- Edit booking page


### Access to pages according to the login status:

| Page Name         | Logged in      | Not Logged In| 
| -------------     | ------------- | -------------| 
| home page         | Y             | Y            |
| about page        | Y             | Y            |
| login page        | Y             | Y            |
| registration page | Y             | Y            |
| logout page       | Y             | N            |
| make booking page | Y             | N            |
| my bookings page  | Y             | N            |
| edit booking page | Y             | N            |

**Make a booking page**

- The Booking page features a form to create a booking. User needs to enter:
  - Full name

  - Phone number
  - Message (optional)
